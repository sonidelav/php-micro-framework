# PHP Micro Framework

A PHP Framework that can be compiled in a single `.php` file

## Installation
```bash
$ composer install
```

## Console
```bash
$ php console
```

### Compile
```bash
# php console app:compile --bootstrap src/app/bootstrap.php [--output=[Output Filename]]
```

### Serve
```bash
$ php console app:serve [--prod]
```
